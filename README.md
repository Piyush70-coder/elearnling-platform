# 🎓 eLearning Platform

A modern, responsive eLearning platform built using **Django** and **Tailwind CSS**.  
This platform allows students to access courses, assignments, and video lectures in a clean and user-friendly interface.

---

## 🚀 Features

- 👤 User Authentication (Login / Register)
- 📚 Course Management
- 🎥 Lecture Video Support
- 📝 Assignment Upload System
- 💬 Discussion Section
- 📁 Media & Static File Handling
- 🎨 Modern UI using Tailwind CSS
- 📱 Fully Responsive Design
